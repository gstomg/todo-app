# Инструкция по деплою на GitHub Pages

## Проблема была решена

Ошибка `404 (Not Found)` возникала из-за неправильной настройки `base` URL в Vite конфигурации.

## Что было исправлено

1. **Обновлен `vite.config.ts`**:
   ```typescript
   base: '/todo-app/', // правильный base для GitHub Pages
   ```

2. **Создан GitHub Actions workflow** (`.github/workflows/deploy.yml`):
   - Автоматическая сборка при push в main
   - Автоматический деплой на GitHub Pages
   - Создание файла `.nojekyll` для правильной обработки

3. **Обновлен `package.json`**:
   - Уже был настроен `homepage` и скрипты для деплоя

## Как деплоить

### Автоматический деплой (рекомендуется)
1. Сделайте commit и push в ветку `main`
2. GitHub Actions автоматически соберет и задеплоит проект
3. Приложение будет доступно по адресу: `https://gstomg.github.io/todo-app/`

### Ручной деплой
```bash
npm run deploy
```

## Настройка GitHub Pages

1. Перейдите в Settings → Pages вашего репозитория
2. В разделе "Source" выберите "GitHub Actions"
3. Убедитесь, что workflow файл присутствует в репозитории

## Проверка

После деплоя проверьте:
- ✅ CSS файлы загружаются корректно
- ✅ JavaScript файлы загружаются корректно
- ✅ Все пути начинаются с `/todo-app/`
- ✅ Приложение работает без ошибок в консоли
